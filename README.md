# CI/CD Project Starter

A beginner-friendly CI/CD project with:
- React frontend
- Node.js/Express backend
- Docker
- Kubernetes manifests
- GitHub Actions pipeline

## Local run

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Docker
```bash
docker compose up --build
```
